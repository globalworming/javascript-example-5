Maps

Maps makes building relationships between objects easier 

const atom = {"name": "atom"};
const thunder = {"name": "thunder"};
const antimatter = {"name": "antimatter", evil: true };
const lightning = {"name": "lightning", evil: true };

const heroesToArchEnemy = new Map();
heroesToArchEnemy.set(atom, antimatter);
heroesToArchEnemy.set(thunder, lightning);
heroesToArchEnemy // -> Entries:
    //  0: {Object => Object} key: {name: 'atom'} value: {name: 'antimatter', evil: true}
    //  1: {Object => Object} key: {name: 'thunder'} value: {name: 'lightning', evil: true}
