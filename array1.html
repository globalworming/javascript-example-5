Arrays

array values don't have to be of the same type 

const array = ["yes", true, 1];
for (const value of array) {
    console.log(typeof value)
}
// -> string, boolean, number
and have multiple ways of getting these values. the values are indexed by numbers, starting at 0 

const array = ["yes", true, 1];
console.log(array[0])
// -> "yes"
console.log(array[2])
// -> 1
// or by string representation of number
console.log(array["1"])
// -> true
you can loop through them in many ways, more common ones are 

// for loop
const array = ["a", "b", "c"]
for (let i = 0; i < array.length; i++) {
    console.log(`${i}: ${array[i]}`)
}



// for-of loop
const array = ["a", "b", "c"]
let i = 0
for (const value of array) {
    console.log(`${i++}: ${value}`)
}
or if you want to make it especially hard for yourself, use the array iterator 

const array = ["a", "b", "c"]
const iterator = array.keys()
let next = iterator.next();
while (!next.done) {
    console.log(`${next.value}: ${array[next.value]}`)
    next = iterator.next()
}
