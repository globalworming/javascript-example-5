Arrays

Javascript has a lot of built-in objects. Well start with three 

const object = Object(); // or const object = {}
const array = Array(); // or const array = []
const map = new Map()
which can look pretty similar because you can think of them as containers for key-value pairs 

const object = { 0: "first value" }
const array = [ "first value" ]
const map = new Map([[0, "first value"]]);

object[0] // -> 'first value'
array[0] // -> 'first value'
map.get(0) // -> 'first value'
Let's start with arrays because they are arguably the simplest
you can construct arrays in multiple ways 

// initializer notation
const array1 = ["a", "b"]
// Array with or without new
const array2 = new Array("a", "b")
const array3 = Array("a", "b")
// Array but length
const array4 = Array(4)
console.log(array4)
// -> [,,,,]
const array5 = []
array5[0] = "hi"
console.log(array5)
// -> ["hi"]
