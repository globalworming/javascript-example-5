Objects

Arrays are objects. Objects have String properties, Arrays are pretty much limited to Strings of positive integers 

const array = ["first value"];
console.log(typeof array)
// -> object
array["1"] = "second value"
const object = {"a property": "a value"}
object.anotherProperty = "another value"
console.log(object)
// -> {a property: 'a value', anotherProperty: 'another value'}
in comparison to arrays, objects aren't really built for iteration over their properties, you can do it though if need be. use for-in loop or get an array with key-value pairs 

for (const property in object) {
  console.log(`${property}: ${object[property]}`);
}
// -> a property: a value
// -> anotherProperty: another value
console.log(Object.entries(object))
/* -> 0: ['a property', 'a value']
      1: ['anotherProperty', 'another value'] */
Objects are for tracking different things with properties. values can be anything, other objects, functions, ...
We haven't talked about functions yet or what 'this' is. Functions are resuable code blocks, 'this' here referring to the object 

const human = {
    heads: 1,
    behead: function () {
        this.heads = 0
    }
}
const hydra = {
    heads: 1,
    behead: function () {
        this.heads++
    }
}
